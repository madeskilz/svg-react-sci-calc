# svg-react-sci-calc

A scientific calculator built with React and SVG.

## Features

- SVG-rendered display and buttons for standard and scientific functions (`sin`, `cos`, `tan`, `sqrt`, `log`, exponentiation).
- Simple, modern UI.
- Evaluates entered expression using safe JavaScript `Math` functions.

## Getting Started

1. Initialize a new React project (for example, [Create React App](https://create-react-app.dev/)).
2. Place `src/App.js` in your `src` directory, replacing or alongside the default App.
3. `npm install` if you haven't already.
4. Run your React app:  
   - For CRA: `npm start`
   - For Vite: `npm run dev`

## To extend

- Add more scientific functions in `safeEval`.
- Improve the SVG visuals/styles.
- Log history or connect to a Node.js backend.
